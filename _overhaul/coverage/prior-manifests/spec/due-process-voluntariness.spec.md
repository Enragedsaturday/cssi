# SPEC — Due-Process Voluntariness of Confessions
slug: `due-process-voluntariness` · bucket: `confessions-id` · status: NEW · template: Rule · Key cases (table: case · holding · weight · CL link) · Nuances & limits · Common pitfalls · Visual (Mermaid if multi-factor) · Flashcards (2-5 ::) · Sources

## Scope
Before (and alongside) Miranda, the Due Process Clause bars involuntary confessions — those produced by coercion that overbears the suspect's will, judged by the totality of the circumstances. Coercion must be POLICE conduct (Connelly). This line still governs voluntariness even when Miranda is satisfied.

  - **Ashcraft v. Tennessee** — _PROGENY — 36 hours of relay interrogation is 'inherently coercive'_ · SCOTUS · `DECK`

## Visual (Mermaid)
Voluntariness = totality (duration, deprivation, threats, suspect's characteristics) AND requires POLICE coercion (Connelly). Overborne will → involuntary → inadmissible.

## Common pitfalls (must cover)
  - thinking Miranda warnings cure an involuntary confession
  - ignoring the police-coercion requirement (Connelly)
  - treating any deception as automatically coercive
