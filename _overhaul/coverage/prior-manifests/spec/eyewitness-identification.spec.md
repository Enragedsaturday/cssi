# SPEC — Eyewitness Identification
slug: `eyewitness-identification` · bucket: `confessions-id` · status: NEW · template: Rule · Key cases (table: case · holding · weight · CL link) · Nuances & limits · Common pitfalls · Visual (Mermaid if multi-factor) · Flashcards (2-5 ::) · Sources

## Scope
Two doctrines: (1) a 6A right to counsel at POST-charge corporeal lineups (Wade/Gilbert; not pre-charge — Kirby; not photo arrays — Ash), and (2) a due-process bar on identifications produced by unnecessarily suggestive procedures that create a substantial likelihood of misidentification, judged by reliability factors (Biggers/Manson). Perry: due-process screening applies only to police-arranged suggestion.
